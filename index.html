Start Up Verde Viviateña

La start up verde yoreciclo.io es una iniciativa tecnológica para ayudar a la ciudadanía de Viviate mediante el reciclaje. Cada ciudadano podrá reciclar cualquier producto, acumular puntos y canjear estos puntos ya sea en bonos de internet o alimentos de primera necesidad necesarios para las familias dada la coyuntura actual.
